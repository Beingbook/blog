## React Context

React Context를 이용하여 상태를 하위 컴포넌트로 직접 `Prop`으로 전달하지 않고 참조하게 할 수 있습니다.

```jsx
const idContext = React.createContext(null);

const FormControl = ({ children, label, id = getSomeUniqueId() }) => (
  <div>
    <label htmlFor={id}>{label}</label>
    <div>
      <idContext.Provider value={id}>{children}</idContext.Provider>
    </div>
  </div>
);

const Input = (props) => {
  const id = React.useContext(idContext) || props.id;
  return <input {...props} id={id} aria-labelledby={id} />;
};

// FormControl과 Input에 id를 전달하지 않아도 됩니다.
const SomeForm = () => (
  // ...
  <FormControl label="사용자 이름">
    <div className="some-beautiful-deco">
      <Input />
    </div>
    <div className="some-error-message">BlahBlahBlah...</div>
  </FormControl>
  // ...
  <FormControl label="사용자 이름">
    <div className="some-beautiful-deco">
      <Input />
    </div>
    <div className="some-error-message">BlahBlahBlah...</div>
  </FormControl>
  // 필요하다면 직접 ID를 넘겨도 됩니다.
  <FormControl label="사용자 이름" id="unique-id">
    <div className="some-beautiful-deco">
      <Input />
    </div>
    <div className="some-error-message">BlahBlahBlah...</div>
  </FormControl>
  // ...
);
```

`getSomeUniqueId` 또한 React Context와 Hook 조합으로 멋지게 만들어 낼 수 있습니다.
